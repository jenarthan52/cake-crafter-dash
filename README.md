# 🎂 Cake Crafter Dash

Cake Crafter Dash is a fun and interactive cake crafting game built with modern web technologies. Players can create, customize, and serve delicious cakes while racing against time to satisfy customers and earn high scores.

## 🚀 Live Demo

🌐 Live Website: https://cake-crafter-dash.lovable.app/

---

## ✨ Features

- 🎨 Beautiful and responsive UI
- 🍰 Interactive cake customization
- ⏱️ Fast-paced gameplay mechanics
- 🏆 Score tracking system
- 📱 Mobile-friendly design
- ⚡ Smooth animations and transitions

---

## 🛠️ Tech Stack

- React.js
- TypeScript
- Tailwind CSS
- Vite
- Lovable Platform
